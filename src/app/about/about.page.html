Sobre o Sabor App Sabor App

Este aplicativo foi criado como Projeto Final da disciplina de Desenvolvimento de Aplicações Móveis.

O Aplicativo foi desenvolvido como uma solução prática para consulta de receitas, utilizando Ionic e Angular integrado a uma API pública. As receitas são carregadas dinamicamente para manter o conteúdo sempre atualizado.

O sistema utiliza o recurso de infinite scroll, permitindo que novas receitas sejam exibidas conforme o usuário rola a tela, garantindo uma navegação fluida e eficiente com apoio dos componentes nativos do Ionic.

Funcionalidades Principais

Busca Avançada de Receitas (API TheMealDB) Modo Offline Completo: Salve suas receitas favoritas no perfil e acesse todos os detalhes (ingredientes, modo de preparo) mesmo sem internet. Perfeito para cozinhas com sinal fraco! Time de Desenvolvimento
